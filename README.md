# 🚀 ColloQ - Mock Interview Platform (Backend)
Developed by **Team RYATH** 🇱🇰

ColloQ is a high-performance mock interview platform built to bridge the gap between candidates and industry experts. The backend is powered by Spring Boot, providing a secure and scalable architecture for booking, payments, and real-time wallet management.

## 🛠️ Tech Stack
* **Framework:** Spring Boot 3.x
* **Language:** Java 17+
* **Security:** Spring Security & JWT (Stateless)
* **Database:** MySQL / PostgreSQL
* **ORM:** Spring Data JPA (Hibernate)
* **Build Tool:** Maven
* **Integrations:** Google Calendar API, Stripe/Stripe-like Payment Gateway

## 🌟 Key Features
* **JWT Authentication:** Secure role-based access for Candidates, Interviewers, and Admins.
* **Smart Wallet System:** Automated 90/10 revenue split between Interviewers and the platform.
* **Booking Engine:** Real-time availability management and meeting link generation.
* **Automated Emails:** Professional interview invitations and feedback reports.

## 🚀 Getting Started
1. Clone the repository.
2. Update `application.properties` with your Database and JWT Secret credentials.
3. Run `mvn clean install`.
4. Start the application on `port 8080`.
